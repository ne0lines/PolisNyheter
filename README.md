# Polisnyheter

En enkel webbapp för att visa polisnyheter från Sveriges Polismyndighet API.

## Setup

1. Installera Node.js från https://nodejs.org/

2. Installera dependencies:
   ```
   npm install
   ```

3. Bygg projektet:
   ```
   npm run build
   ```

4. Öppna index.html i en webbläsare eller använd en lokal server.

## Användning

Appen hämtar och visar polisnyheter från https://polisen.se/api/events.
